# md-editor

### Original repository is [here](https://github.com/voraciousdev/vue-markdown-editor)

## Contribute

This library uses `yarn`.

### Install dependencies

```shell
yarn install
```

### Compile and hot-reload for development

```shell
yarn serve
```

### Compile for production

```shell
yarn build
```

### Run the unit tests

```shell
yarn test:unit
```

### Lint and fix files

```shell
yarn lint
```
