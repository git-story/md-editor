# md-editor

### 원본 레포지토리는 [여기](https://github.com/voraciousdev/vue-markdown-editor)에서 확인하실 수 있습니다.

## 빌드

### 종속성 설치

```shell
npm install
```

### Vue 테스트 서버 실행

```shell
npm run serve
```

### 배포용 빌드

```shell
npm run build
```

### 테스트 실행

```shell
npm run test:unit
```

### Lint

```shell
npm run lint
```
